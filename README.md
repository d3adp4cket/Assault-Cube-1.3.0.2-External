# Assault Cube 1.3.0.2 External

A lightweight, purely external C# trainer for Assault Cube 1.3.0.2.

![Menu Preview](Screenshot.png)

## Features

- **God Mode**: Automatically locks Health and Armor completely, keeping you alive indefinitely.
- **Infinite Ammo**: Rapidly replenishes all weapons ammo every tick, never run out during a firefight.
- **Fast Fire**: Nullifies weapon fire delay, enabling instant cyclic rate for all firearms (AR, Sniper, Shotgun).

## Building

Requires the [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0).

Navigate to the project directory and invoke the build script or use the `dotnet` CLI directly:

```console
$ dotnet build -c Release
```

## Usage

1. Open `Assault Cube 1.3.0.2`.
2. Launch `Assault Cube 1.3.0.2 External.exe`.
3. The trainer will automatically attach and hook into the games memory space. 
4. Select your tools via the toggles. Enjoy.
